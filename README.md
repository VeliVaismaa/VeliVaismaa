# Hei — moi — hello 👋

Olen **Veli Vaismaa** — oululainen luonto-opas, muusikko ja FM (tietojenkäsittely). Toiminnassani luonto, kulttuuri ja paikkatieto kohtaavat: **[OuluNature.fi](https://oulunature.fi)** (luonto-opastus ja hyvinvointiretket) ja **[VeliVaismaa.fi](https://velivaismaa.fi)** (musiikki, perinnekupletit, yhteisötaide).

GitHub on minulle kirjasto — täältä löydän omaan toimintaan elävät pienet sovellukset, paikkatietokokeilut välineet, jotka tukevat retkien suunnittelua ja omaa työtä. AI ja Claude Code liitännäisineen on isossa työkalun roolissa.

I'm **Veli Vaismaa** — a nature guide, musician and MSc (Computer Science) based in Oulu, Finland. In my work nature, culture and geospatial meet: **[OuluNature.fi](https://oulunature.fi)** (guided nature walks and wellbeing trips) and **[VeliVaismaa.fi](https://velivaismaa.fi)** (music, traditional Finnish *kupletti* songs, community arts).

GitHub is a library and toolbox for me — small apps, geospatial experiments and other tools that support trip planning, species identification, and music work. AI and Claude Code have opened up a new world.

### What I do

**Guided nature trips around Oulu** — bird walks (Liminganlahti, Hietasaari), mushroom forays, urban nature and wellbeing.
**Music and tradition** — community sing-alongs, care home concerts, the *Oulu's Lost Kupletti Songs* archival project (reviving unpublished local songs from the 1940s–1990s)
**Maps and geospatial work** — QGIS trip maps, species observation visualisations, FinBIF API tools
**Mindfulness facilitation** — strength-based presence practice integrated into nature trips
